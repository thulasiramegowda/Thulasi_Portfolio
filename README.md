# The Journey



A personal engineering portfolio documenting my academic journey, projects, certifications, skills, and milestones.



## Live Website



Vercel: https://thulasi-portfolio-21.vercel.app/ 



## Local Development



```bash

npm install

node server.js

```



Open:



```text

http://localhost:3000

```
