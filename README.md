# The Journey

A personal engineering portfolio documenting my academic journey, projects, certifications, skills, and milestones throughout my B.Tech in Computer Science and Information Technology.

## About

This portfolio serves as a living record of my growth as a student, developer, and aspiring entrepreneur. It showcases my projects, academic progress, certifications, and learning journey in technology.

## Current Status

The project runs successfully in a local development environment.

Due to the current backend architecture (Node.js, Express.js, local JSON database, and file upload system), deployment to Vercel is not yet configured. I am actively learning deployment workflows and plan to deploy an improved version in the future.

## Run Locally

```bash
npm install
node server.js
```

Open:

```text
http://localhost:3000
```
